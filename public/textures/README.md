# Texture Drop Folder

Place your custom block textures in this folder using these filenames:

- `dirt.png`
- `stone.png`
- `log_side.png`
- `log_top.png`
- `leaves.png`
- `water.png`
- `grass_top.png`
- `sand.png`
- `snow.png`
- `ice.png`
- `swamp_grass.png`
- `swamp_log_side.png`
- `swamp_log_top.png`
- `swamp_leaves.png`
- `cactus_side.png`
- `cactus_top.png`
- `swamp_reed.png`
- `shrub.png`
- `tall_grass.png`
- `sedge.png`

Gooey mob textures (optional):

- `gooey.png` (single texture fallback for all faces)
- `gooey_side.png`
- `gooey_top.png`
- `gooey_bottom.png`
- `gooey_front.png`
- `gooey_back.png`
- `goo_puddle.png` (trail puddle texture)

Legacy fallback names still supported for the current gooey mob:

- `mob.png`, `mob_side.png`, `mob_top.png`, `mob_bottom.png`, `mob_front.png`, `mob_back.png`

Recommended format:

- PNG
- 16x16 (or 32x32, 64x64 if you prefer)
- pixel art style (nearest-neighbor filtering is enabled)

Notes:

- If a file is missing, the game uses a generated placeholder texture.
- You can replace files and refresh to test quickly.
